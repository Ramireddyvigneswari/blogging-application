#  Blogging Application

## Overview

This is a RESTful Blogging Application developed using Spring Boot.
It allows users to create, manage, and retrieve blog posts.

---

## 🛠 Tech Stack

* Java 17
* Spring Boot
* Spring Data JPA
* PostgreSQL
* Hibernate
* Postman

---

##  Features

* Create Users
* Create Categories
* Create Blog Posts
* REST API Architecture
* Database Integration

---

##  API Endpoints

###  User

* POST `/api/users/`
* GET `/api/users/`

###  Category

* POST `/api/categories/`

###  Post

* POST `/api/posts/`
